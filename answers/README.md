# Concept & Design Answers

Prose answers to the tutor's **Concept / Design Q** drills (Micro-Tutor mode 9) — MRO walkthroughs,
"ABC vs Protocol and why", "why are default args evaluated once", and similar explain-in-your-own-words
prompts that don't require code.

One file per topic, e.g. `mro-c3-linearization.md`, `abc-vs-protocol.md`. The tutor grades the written
explanation against the study-guide reasoning bar and logs the verdict in `Custom-Work/TUTOR_STATE.json`
under `concept_answers`.
